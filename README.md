## Key Bindings
* Toggle Fire Mode: "Caps Lock"
* Use Grenade: "G"
* Use First Aid Kit: "H"
* Use Bandage: "B"
* Use Boost Item: "T"
* Map: "`"
* Mark My Location: "Ctrl + Space Bar"
